# Physics_Nasa
## Requirements
### Server
install express
```
npm install express
```
nodemon active project
```
npm install --save-dev nodemon
```

### Client
install react dependencies
```
npm install -y
```
start the react app
```
npm start
```

### All together scripts
#### from the root directory: `/physics_nasa`
install client&server
```
npm run install-all
```
run all
```
npm run watch
```
